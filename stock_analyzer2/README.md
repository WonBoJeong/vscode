## 🎯 사용법 (개선된 UI로 더 편리하게)

### **1단계: 원클릭 자동 설정**
1. **종목 선택**: 16개 사전 설정 버튼 중 하나 클릭
2. **Download & Auto Setup** 버튼 클릭
3. **자동 완료**: 데이터 다운로드 + 평단가/보유량 자동 설정

### **2단계: 향상된 가독성으로 투자 분석**
1. **📈 Analyze** 버튼 클릭
2. **4개 패널에서 더 읽기 쉬운 정보 확인**:
   - **📊 Stock Info**: 종목 기본 정보 (기존 폰트 유지)
   - **💼 Position**: 평단가, 보유량, 평가손익 (기존 폰트 유지)
   - **🔍 Technical**: 3일평균, 95% 신뢰구간 (기존 폰트 유지)
   - **🎯 Trading Signal**: 매수/매도 신호 (기존 폰트 유지)

### **3단계: 개선된 폭락 대응 전략 (핵심 개선)**
1. **Crash Strategy 탭** 이동 → **14pt 굵은 제목으로 명확함**
2. **최적 손절가 계산** 클릭
3. **13pt 폰트로 읽기 쉬운 4가지 전략 상세 분석표** 확인
4. **12pt 폰트 AI 자문 리포트** 생성하여 전문가 상담

### **4단계: 개선된 투자 계산 (Investment 탭)**
1. **Total Budget**: 자동 설정된 예산 확인/수정
2. **Strategy**: Single/DCA/Pyramid 선택
3. **Calculate**: **12pt 폰트로 읽기 편한 투자 계획** 자동 계산
